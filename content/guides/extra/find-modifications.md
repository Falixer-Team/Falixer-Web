## 简介

Pterodactyl 主题和插件可以从多个渠道获取，但并非所有来源都值得信任。下文整理了常见市场及其优缺点，并说明从未验证来源下载修改包的风险。

## Falixer 扩展列表

> 寻找可信扩展时，建议优先使用 [Falixer 扩展列表](/browse)。其中收录了经过 Falixer 管理员验证的扩展页面。

[![Falixer 扩展列表中的扩展预览。](/img/guides/nebula_browse.jpeg)](/browse/nebula)

该列表提供多种筛选条件，以及其他来源可能没有的统计信息。目前尚不能直接在站内下载或购买扩展，相关功能计划在未来加入。

| 优点 | 缺点 |
| --------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 便于寻找新发布或成熟的 Pterodactyl 主题和插件。 | 目前还不能直接通过 Falixer 网站购买或下载扩展，相关功能计划后续提供。 |
| 扩展提交会按照更严格的要求由 Falixer 团队审核。 |  |
| 提供多种免费的 Pterodactyl 主题与插件。 |  |
| 由 Falixer Pterodactyl 扩展框架的同一组织运营。 |  |
| 可以查看某个扩展当前被多少个 Pterodactyl 面板使用。 |  |

[**前往 Falixer 扩展列表寻找扩展**](/browse)

## sourceXchange

[sourceXchange](https://sourcexchange.net) 是 Pterodactyl 插件和主题的常见来源，也是 Pterodactyl 社区中较受认可的市场。

[![sourceXchange 上的 Falixer 扩展预览。](/img/guides/nebula_sourcexchange.jpeg)](https://sourcexchange.net/products/nebula)

sourceXchange 提供多种筛选条件，也包含 Paymenter 主题与插件、Pterodactyl Egg、Minecraft 插件与地图、WHMCS 模块与主题等分类。

| 优点 | 缺点 |
| ---------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------ |
| 管理员通常会验证商品合法性，并审核商品源代码。 | 仅支持 PayPal 和 Stripe，本地常用支付渠道可能不可用。 |
| 拥有较丰富的 Pterodactyl 插件和主题资源。 | Falixer 扩展可能不会遵循 [Falixer 扩展列表](/browse)的同等质量要求。 |
| 已购授权的商品更新时会发送邮件通知。 |  |
| 可以访问通过 sourceXchange 分发的历史版本。 |  |

总体而言，sourceXchange 是购买 Pterodactyl 产品授权的常用平台。

## BuiltByBit

[BuiltByBit](https://builtbybit.com?ref=581299) 是规模较大的市场。由于商品数量庞大，部分损坏或质量较低的 Pterodactyl 主题、插件及 Falixer 扩展也可能通过审核。

[![BuiltByBit 上的 Falixer 扩展预览。](/img/guides/nebula_builtbybit.jpeg)](https://builtbybit.com/resources/nebula.32442/?ref=581299)

该平台支持更多本地支付渠道，也可以一次结算多个商品，但会向创作者收取约 10% 的平台费用。

| 优点 | 缺点 |
| ----------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 商品种类和数量较多。 | 网站底层使用较旧的论坛软件，访问体验有时较慢。 |
| 支持本地及加密货币支付，适合无法使用信用卡或借记卡的用户。 | 部分低质量商品可能未经源代码审核便进入平台。 |
|  | 严格禁止多账户；使用动态 IP 时，同一 IP 下登录可能导致账号被误判限制。 |

## 远离不可信来源

部分不可信网站会未经授权重新分发 Pterodactyl 相关商品。这些网站通常缺乏管理和源代码审核，下载文件中可能被植入恶意或不可信代码。请仅从可信渠道获取扩展，并在安装前核验来源。
